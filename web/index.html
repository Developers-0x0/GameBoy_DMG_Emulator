Game Boy DMG-01 Emulator

Select a Game Boy ROM file to start playing

Controls

Arrow Keys: D-Pad
Z: A Button
X: B Button
Enter: Start
Space: Select
R: Reset Game

Instructions

  1. Select a Game Boy ROM file (.gb or .gbc)
  2. Click "Start Game" to begin emulation
  3. Use the keyboard controls to play
  4. Use "Pause" to pause/resume the game
  5. Use "Reset" to restart the current game
